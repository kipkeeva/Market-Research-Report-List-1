<p>&nbsp;</p>
<p><p>Thermo-responsive shape memory polymer (SMP) market can be categorized into two types - Type I and Type II. Type I refers to SMPs that exhibit a reversible shape memory effect solely based on temperature changes, with the polymer reverting to its original shape upon heating. Conversely, Type II SMPs exhibit shape memory effect not only with temperature changes but also in response to additional stimuli such as light, electricity, or pH. These SMP types offer diverse applications in fields like biomedicine, aerospace, textiles, and automotive due to their ability to change shape and recover their original form.</p></p>
